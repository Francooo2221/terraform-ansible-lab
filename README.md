# Terraform + Ansible DevOps Lab

## Opis projektu
Mini-lab DevOps pokazujący, jak stworzyć i skonfigurować środowisko w chmurze przy użyciu Infrastructure as Code (IaC) i Configuration as Code (CaC).

Projekt obejmuje:
- **Terraform**: tworzenie maszyny wirtualnej (VM) w Google Cloud Platform (GCP), konfigurację VPC i firewall.  
- **Ansible**: konfiguracja VM, instalacja i uruchomienie serwera `nginx`.

## Technologie
- Terraform + GCP  
- Ansible + GCP  
- Bash / Linux  
- Ubuntu Linux  

## Zdobyte umiejętności
- Tworzenie i zarządzanie zasobami w Google Cloud Platform przy użyciu Terraform.  
- Automatyzacja konfiguracji i wdrażania serwerów z Ansible.  
- Projektowanie i utrzymywanie infrastruktury jako kod (IaC) w powtarzalny sposób.  
- Bezpieczne zarządzanie kluczami SSH i konfiguracja dostępu do środowisk chmurowych.  
- Wdrażanie i konfiguracja serwera webowego (nginx) w chmurze.
